# Formatação de Trabalhos Acadêmicos no Microsoft Word (Normas ABNT)

## Introdução

Durante a vida escolar e acadêmica é comum produzir **trabalhos escritos** para apresentar pesquisas, análises ou projetos.

Para que esses trabalhos tenham **organização e padronização**, existem normas técnicas que orientam como o documento deve ser estruturado.

No Brasil, essas normas são definidas pela **ABNT (Associação Brasileira de Normas Técnicas)**.

Essas normas são utilizadas em diversos tipos de trabalhos, como:

- Trabalhos escolares
- Relatórios técnicos
- Artigos científicos
- Trabalhos de conclusão de curso (TCC)
- Dissertações e teses

Seguir essas normas garante:

- organização do documento  
- padronização entre trabalhos  
- facilidade de leitura  
- apresentação profissional  

Neste material vamos aprender **como estruturar e configurar um trabalho acadêmico no Microsoft Word** seguindo essas normas.

---

# Estrutura de um Trabalho Acadêmico

Um trabalho acadêmico é dividido em **três grandes partes**:

1. **Elementos Pré-Textuais**
2. **Elementos Textuais**
3. **Elementos Pós-Textuais**

Cada parte possui uma função específica dentro do documento.

---

# Elementos Pré-Textuais

Os **elementos pré-textuais** são todas as partes que aparecem **antes do conteúdo principal do trabalho**.

Eles servem para **identificar o documento e organizar as informações iniciais**.

Os principais elementos pré-textuais são:

- Capa  
- Folha de rosto  
- Sumário  
- Lista de figuras  
- Lista de tabelas  

---

# Capa

A **capa** é a primeira página do trabalho.

Ela apresenta as informações básicas que identificam o documento.

Normalmente a capa contém:

- Nome da instituição  
- Nome do autor (aluno)  
- Título do trabalho  
- Cidade  
- Ano  

A capa permite identificar rapidamente **quem produziu o trabalho e qual é o tema apresentado**.

---

# Folha de Rosto

A **folha de rosto** é a página que vem logo após a capa.

Ela apresenta informações semelhantes à capa, mas inclui também **uma descrição do trabalho**.

Exemplo de descrição:

> Trabalho apresentado à disciplina de Tecnologia da Informação como requisito parcial para avaliação.

A folha de rosto explica **em qual contexto o trabalho foi produzido**.

---

# Sumário

O **sumário** é uma lista organizada das partes do trabalho.

Ele apresenta:

- títulos das seções
- subtítulos
- número das páginas

Exemplo simplificado:

1 Introdução .................. 3 2 Desenvolvimento ............. 5 3 Conclusão ................... 10

O sumário ajuda o leitor a **encontrar rapidamente qualquer parte do trabalho**.

No Microsoft Word, o sumário pode ser **gerado automaticamente** quando utilizamos os estilos de título.

---

# Lista de Figuras

A **lista de figuras** apresenta todas as imagens utilizadas no trabalho.

Essas imagens podem ser:

- gráficos
- fotografias
- diagramas
- ilustrações
- capturas de tela

Essa lista permite que o leitor **localize facilmente as imagens dentro do documento**.

---

# Lista de Tabelas

A **lista de tabelas** apresenta todas as tabelas utilizadas no trabalho.

Tabelas organizam informações em **linhas e colunas**, facilitando a leitura de dados.

Assim como a lista de figuras, ela ajuda o leitor a **encontrar rapidamente determinadas informações no documento**.

---

# Elementos Textuais

Os **elementos textuais** são o conteúdo principal do trabalho.

É nessa parte que o autor apresenta ideias, análises e explicações sobre o tema estudado.

Normalmente essa parte é dividida em três seções principais:

- Introdução  
- Desenvolvimento  
- Conclusão  

---

# Introdução

A **introdução** é o início do texto.

Nela o autor apresenta:

- o tema do trabalho  
- o objetivo da pesquisa  
- uma breve explicação sobre o assunto  

A introdução ajuda o leitor a entender **sobre o que será o trabalho**.

---

# Desenvolvimento

O **desenvolvimento** é a parte principal do trabalho.

Nessa seção são apresentados:

- conceitos
- explicações
- análises
- exemplos
- resultados de pesquisas

É nessa parte que o autor **explora o tema com maior profundidade**.

---

# Conclusão

A **conclusão** é a parte final do texto.

Nela o autor apresenta:

- os resultados obtidos
- as principais conclusões
- reflexões finais sobre o tema

A conclusão encerra o trabalho retomando **os objetivos apresentados na introdução**.

---

# Elementos Pós-Textuais

Os **elementos pós-textuais** aparecem após o conteúdo principal do trabalho.

Eles complementam as informações apresentadas.

Entre os principais estão:

- Referências  
- Anexos  
- Apêndices  

---

# Referências

As **referências** apresentam a lista de todas as fontes utilizadas para produzir o trabalho.

Essas fontes podem ser:

- livros
- artigos científicos
- sites
- documentos
- vídeos

Incluir referências é importante porque:

- mostra de onde vieram as informações
- dá credibilidade ao trabalho
- evita plágio

---

# Anexos

Os **anexos** são documentos adicionais que **não foram produzidos pelo autor**, mas ajudam a complementar o trabalho.

Exemplos:

- leis
- documentos oficiais
- tabelas externas
- imagens de outras fontes

---

# Apêndices

Os **apêndices** são materiais complementares **produzidos pelo próprio autor do trabalho**.

Exemplos:

- questionários aplicados em uma pesquisa
- formulários criados pelo autor
- materiais desenvolvidos durante o estudo

---

# Configuração do Documento no Word

Antes de começar a escrever um trabalho acadêmico, é importante configurar corretamente o documento no Word.

As principais configurações são:

- tamanho do papel
- margens
- fonte
- espaçamento
- parágrafos

---

# Tamanho do Papel

O padrão utilizado em trabalhos acadêmicos é o **papel A4**.

Dimensões do papel A4:

21 cm x 29,7 cm

No Word:

Layout → Tamanho → A4

---

# Margens do Documento

Segundo as normas da ABNT, as margens devem ser configuradas da seguinte forma:

Margem superior: 3 cm Margem esquerda: 3 cm Margem inferior: 2 cm Margem direita: 2 cm

No Word:

Layout → Margens → Margens Personalizadas

---

# Fonte do Documento

As fontes mais utilizadas em trabalhos acadêmicos são:

- Arial
- Times New Roman

Tamanho padrão:

12

Essa configuração garante **boa leitura do texto**.

---

# Alinhamento do Texto

O texto deve ser **justificado**.

Isso significa que o texto fica alinhado **dos dois lados da página**, criando uma aparência mais organizada.

No Word:

Página Inicial → Justificar

---

# Espaçamento entre Linhas

O espaçamento padrão entre linhas deve ser:

1,5

Esse espaçamento melhora a leitura e permite espaço para eventuais correções ou anotações.

---

# Parágrafo

Cada novo parágrafo deve ter **recuo na primeira linha**.

Padrão utilizado:

1,25 cm

Isso ajuda a identificar visualmente o início de um novo parágrafo.

---

# Exercício da Aula

Crie um documento no Microsoft Word e faça as seguintes configurações:

1. Definir papel **A4**
2. Ajustar **margens no padrão ABNT**
3. Definir **fonte Arial tamanho 12**
4. Definir **espaçamento entre linhas 1,5**
5. Definir **alinhamento justificado**

Depois crie as seguintes seções:

1 Introdução

2 Desenvolvimento

3 Conclusão

Cada seção deve conter **três parágrafos de texto**.
